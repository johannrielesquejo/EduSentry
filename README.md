📘 EduSentry

EduSentry is an academic project developed for IT Elective 1 and Data Structures and Algorithms, showcasing the application of DSA concepts like Binary Search in a real-world system. A web-based platform designed to enhance educational security, monitoring, and management. Beyond its core system features, EduSentry integrates Data Structures and Algorithms (DSA) concepts to optimize performance, improve data handling, and provide efficient system operations.

🌐 Live Website:
👉 http://edusentry-0jh6.onrender.com

🚀 Features:
🔐 Secure user authentication and access control
📊 Dashboard for monitoring educational data
🧑‍🏫 User management (students, faculty, admin)
📁 Data tracking and record management
⚡ Responsive and user-friendly interface

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript
Backend: Node.js / Express
Database: XAMPP / MySQL / PHP
Deployment: Render
